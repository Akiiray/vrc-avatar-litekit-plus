VRC Avatar LiteKit Plus

メニュー: Tools > VRC Avatar LiteKit Plus > Open LiteKit Window

VRC Avatar LiteKit Plus は、VRChat アバターの軽量化診断、Performance Rank 目標への到達支援、Quest / Android / iOS 向け変換支援を目的とした非破壊系 Unity Editor 拡張です。

現時点では初期開発版であり、本格的な Texture Atlas 化、UV 変更、Mesh 結合、ポリゴン削減、Material 変換は実装していません。

初期方針:

* Dry Run とログ出力を優先します。
* 元 Avatar / Prefab / Material / Texture / Mesh を直接変更しません。
* NDMF 連携は将来機能として Placeholder に留めています。
* 破壊的操作は、将来実装する場合でも明示的な確認なしに実行しない設計にします。
