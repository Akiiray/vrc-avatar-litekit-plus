# Mobile Conversion Notes

Mobile conversion support should begin as recommendations and dry-run reports.

## Targets

- Quest / Android VRChat avatars.
- Android handheld constraints where applicable.
- iOS-related planning if supported by future VRChat workflows.

## Initial scope

- Texture size recommendations.
- Material count and texture memory summaries.
- Performance Rank target planning.

## Out of initial scope

- Automatic shader conversion.
- Automatic material rewriting.
- Mesh simplification.
- Deleting unsupported components.
